# HASS Online Learning Program

A lesson-research agent for South Australian Year 7-10 Humanities and Social
Sciences (HASS). Give it a topic, and it researches the web for current,
credible material and writes out a ready-to-use online lesson (readings,
activities, vocabulary, and a short quiz) as a self-contained HTML page.

South Australian government, Catholic and independent schools teach the
**Australian Curriculum v9** for Years 7-10 HASS (there's no separate SA
syllabus at this level, unlike SACE in Years 11-12).

**Current scope:** Year 9 Geography only (`biomes-and-food-security` and
`geographies-of-interconnections`), as a starting point to prove out the
approach before expanding to other year levels and strands (History, Civics
and Citizenship, Economics and Business).

## How it works

The script (`src/generate_lesson.py`) calls the Claude API with the built-in
`web_search` tool switched on, so the model researches your topic before
writing anything — it's instructed to prefer government, education and
reputable news sources (see `src/prompts.py`), and every lesson ends with a
"Sources used in this lesson" section listing what it found.

**Always have a teacher review a generated lesson — including the sourced
links and any factual claims — before using it with a class.** This is a
drafting tool, not a publishing pipeline.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # then edit .env and add your ANTHROPIC_API_KEY
```

Get a key at https://console.anthropic.com/settings/keys.

## Usage

See what's available to build a lesson around:

```bash
python src/generate_lesson.py --list-descriptors
```

Generate a lesson:

```bash
python src/generate_lesson.py \
  --topic "Desertification and food security in the Sahel" \
  --unit biomes-and-food-security \
  --duration 70
```

This writes two files into `lessons/`:

- `<date>-<slug>.html` — the lesson page, open it in a browser
- `<date>-<slug>.json` — the same content as structured data, in case you
  want to edit it and re-render (`python src/render.py <file.json>`)

`--unit` is either `biomes-and-food-security` or
`geographies-of-interconnections`. `--out` lets you set the output filename
stem instead of deriving it from the topic.

## Example output

`examples/sample-lesson.json` / `examples/sample-lesson.html` is a
hand-checked sample lesson (not generated by the script, since that needs a
live API key) showing the expected output shape and quality bar, on the
Murray-Darling Basin and food security.

## Project layout

```
src/
  generate_lesson.py   CLI entry point / the agent loop
  curriculum.py         Year 9 Geography content-descriptor reference data
  prompts.py            system + user prompt construction
  render.py              renders the model's JSON output into the HTML page
lessons/                 generated output (git-ignored)
examples/                 one hand-checked sample lesson
```

## Extending

- **More strands/year levels:** add entries to `curriculum.py` and a
  `--year`/`--subject` option; the prompt-building and rendering code is
  already generic.
- **A different output format** (Markdown lesson plan, Google Slides): swap
  out `render.py` for a different renderer; the JSON lesson schema in
  `src/prompts.py` (`LESSON_JSON_SCHEMA`) doesn't need to change.

## A note on the curriculum data

`src/curriculum.py`'s content-descriptor codes were compiled from training
knowledge and public search results, not scraped live from
https://v9.australiancurriculum.edu.au (it returns 403 to automated
fetches). Spot-check codes against the official site or your school's
curriculum documents before reporting against them formally.
